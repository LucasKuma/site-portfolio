Xadrez via Terminal

Projeto criado durante um curso de Java do professor Nelio Alves na Udemy. Foi criado um jogo de xadrez para ser jogado no próprio terminal. O jogo respeita todas as regras do xadrez, incluindo as jogadas especiais existentes, que alteram um pouco as regras padrões do jogo.
